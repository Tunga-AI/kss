# **App Name**: KSS Institute Portal

## Core Features:

- Public Course Catalog: Display available courses, institute information, and registration details.
- Learner Dashboard: Provide access to course materials, assignment submissions, progress tracking, and grades.
- Staff/Admin Portal: Manage courses, content, learner enrollment, assignment grading, and generate reports.
- Content Management: Allow staff to create, edit, and organize course content (text, video, files).
- Secure User Authentication: Secure user authentication and authorization to ensure only valid users access different parts of the application.
- AI Course Recommendation Tool: LLM that offers personalized course recommendations based on learner profiles and learning history, to help learners find new and interesting sales training programs

## Style Guidelines:

- Primary color: Professional blue (#2962FF) to convey trust and stability.
- Background color: Light blue (#E6F0FF) for a clean and professional look.
- Accent color: Green (#32CD32) for key actions and highlights.
- Body font: 'PT Sans', a humanist sans-serif.
- Headline font: 'Space Grotesk', a proportional sans-serif.
- Use professional and consistent icons relevant to sales training and learning concepts.
- Clean and organized layout with clear sections for course information, learner dashboard, and staff portal.
